# nepo-backend

## Start node
```bash
npm build
npm start
```
or
```bash
npm build
npm run dev
```

Node app will start in localhost:3000

## GET Endpoints
```bash
/ping
```
```bash
/buyers
/buyer?id=123456
/buyer?id=654321
```
```bash
/sellers
/seller?id=123456
/seller?id=654321
/seller?id=000000 (seller banned)
```
